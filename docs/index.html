Seoul R Meetup

국내 최대 데이터 과학 R 미트업

서울 R 미트업에 오신 것을 환영합니다! 서울 R 미트업은 데이터 사이언스와 관련된 여러 기술을 탐색하고자 하는 분들을 위한 이상적인 장소입니다. R은 데이터 수집, 처리, 탐색적 데이터 분석, 통계 모델링, 시각화 및 기계 학습을 위한 강력한 오픈 소스 소프트웨어이자 프로그래밍 언어로, 쿼토(Quarto)와 함께 전 세계 데이터 과학자들에게 사랑받고 있습니다.

이번 미트업에서는 R 뿐만 아니라 인공지능, 파이썬, SQL 등 데이터 과학의 다른 중요한 구성 요소들도 집중적으로 다룰 예정입니다. 인공지능과 기계 학습은 현대 데이터 분석의 핵심 요소이며, R과 함께 사용될 때 더욱 강력한 통찰력을 제공합니다. 파이썬은 그 유연성과 광범위한 라이브러리로 인해 데이터 사이언스 분야에서 빠르게 성장하고 있으며, R과의 결합을 통해 다양한 데이터 분석 작업을 수행할 수 있습니다.

국내외 저명한 연사들의 강연을 통해 최신 데이터 사이언스 트렌드와 기술에 대한 통찰을 얻을 수 있을 것입니다. 또한, 이 자리는 동료 R 사용자들과 네트워킹을 하고, 지식과 경험을 공유하는 데에도 훌륭한 기회가 될 것입니다.

1 ’24년 미트업 주제

올해 서울 R 미트업의 중심 주제는 ’인공지능과 데이터 사이언스’입니다. 데이터 과학과 인공지능 최신 동향을 탐구하고, 쿼토(Quarto) 활용법도 심도 있게 다루게 됩니다.

2 발표자와 후원회원 모집

  • ’24년 발표자 모집
  • R 사용자회 회비 납부 및 회원 가입

공익법인 한국 R 사용자회 주관으로 R Consortium R User Groups에 등록된 Seoul R Meetup은 매월 개최되는 데이터 사이언스 미트업으로 인공지능, 파이썬, 챗GPT, Data Analytics, Tidyverse, Tidymodels, Shiny, Dashboard, Machine Learning, Digital Writing, Quarto/RMarkdown, Deep Learning 등 통계, AI/ML/DL, 데이터 사이언스, 시각화, 제품개발 등을 실제 현장에서 직접 데이터를 다루고 학교와 연구소에서 연구개발을 하시는 분들 뿐만 아니라 관심있는 분들이 함께 모여 교류하고 지식을 나누는 밋업이니다. 발표를 원하는시는 분 누구나 발표가능하고, 25분 전후의 발표와 5분정도 질의응답으로 진행됩니다. 발표를 원하시는 분은 아래 양식에 맞춰 제출해 주시면 발표 안내를 위해 연락드리도록 하겠습니다.

2024년은 교보생명보험 그룹 데이터 분석 법인 디플래닉스의 후원을 받아 강남 교보타워에서 매월 진행할 예정이고 발표자 및 기여자분들께는 소정의 강사료도 지급될 예정입니다.

2017년부터 시작된 데이터 사이언스 미트업은 특정 주제를 갖고 심화된 주제를 가지고 미트업을 진행하기로 하면서 2021년은 오픈통계패키지(BitStat), 2022년은 디지털 글쓰기(Quarto), 2023년은 챗GPT와 데이터 사이언스, 2024년은 인공지능과 데이터 사이언스를 주제로 진행됩니다. 인공지능 챗GPT가 아니더라도 전통적인 통계, 데이터 사이언스, 딥러닝 등 연관 주제로 지식나눔을 해주실 발표자분들을 기다리고 있습니다.

발표자 모집 등록

공익법인 한국 R 사용자회(이하 R 사용자회)는 2022년 2월 28일 설립되어 창립 1년이 되었습니다. R 사용자회는 R Consortium R User Groups에 등록된 Seoul R Meetup은 매월 개최되는 데이터 사이언스 R 미트업을 2017년부터 운영하며 R 지식 나눔과 커뮤니티 멤버들 간의 사랑방 역할도 수행하고 있습니다.

R 사용자회는 2022년부터 오픈 통계 패키지(BitStat) 패키지를 개발하여 보급하였을 뿐만 아니라 BitNLP, BitSpatial, BitReport 패키지를 개발하여 공개하였을 뿐만 아니라, “디지털 글쓰기”, “데이터 과학 프로그래밍”, “데이터 시각화”, “데이터 과학 기본기”, “오픈 통계 패키지”, “R 텍스트 마이닝”, “데이터 과학 언플러그드” 등 전자책을 집필하여 무료로 대중에 공개하였습니다.

R 사용자회는 출판사 자격도 획득하여 2023년 쿼토 기반 “챗GPT 유닉스 쉘” (전자)책을 출판하여 출판의 새로운 지평을 열었으며 2023년 12월 말 공익법인 인가를 받았습니다. R 사용자회는 2024년에 “인공지능과 데이터 사이언스”를 주제로 매월 데이터 사이언스 미트업을 개최할 예정입니다.
